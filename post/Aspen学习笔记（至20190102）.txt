1、常用模块简介<br />&sect; 混合/分流器
<ul>
<li>Mixer——混合器，把多股物流混合成一股物流
</li>
<li>FSplit——分流器，把一股或多股物流混合后分成多股物流</li>
</ul>
&sect; 调节器
<ul>
<li>Mult——物流倍增器，将物流比例放大或缩小（只需设定缩放因子 Multiplication factor）
</li>
<li>Dupl——物流复制器，将物流复制成任意数量的出口物流（出口可接N股物流）</li>
</ul>
&sect; 闪蒸分离器
<ul>
<li>Flash2——两相闪蒸器，用严格汽-液平衡或汽-液-液平衡，把进料分成两股出口物流。如闪蒸器、蒸发器、分液罐
</li>
<li>Flash3——三相闪蒸器，用严格汽-液-液平衡，把进料分成三股出口物流。如分相器、有两个液相的单机分离器
</li>
<li>Decanter——液-液分相器，把进料分成两股液相出口物流，如分相器、有两个液相而无气相的单极分离器
</li>
<li>Sep——组分分离器，根据规定的组分流率或分率，把入口物流分成多股出口物流，组分分离操作，不考虑分离过程，如蒸馏和吸收
</li>
<li>Sep2——两出口组分分离器，根据规定的组分流率、分率或纯度，把入口物流分成两股出口物流，组分分离操作，不考虑分离过程，例如蒸馏和吸收</li>
</ul>
&sect; 流体输送单元
<ul>
<li>Pump——当已知压力、功率或特性曲线时，改变物流压力。如泵和水轮机
</li>
<li>Compr——压缩机或涡轮机，当已知压力、功率或特性曲线时，改变物流压力。如多变压缩机、多变正排量压缩机、等熵压缩机和等熵涡轮机
</li>
<li>MCompr——多级压缩机或涡轮机，通过级间带有中间冷却器的多级压缩改变物流压力，可从中间冷却器采出液相物流。如多级多变压缩机、多级多变正排量压缩机、多级等熵压缩机和多级等熵涡轮机
</li>
<li>Valve——阀门，确定压降或阀系数，如球阀、蝶阀和截止阀中的多想绝热流动
</li>
<li>Pipe——单管段，计算通过单管段或环形空间的压降或传热量。如恒定直径的管线（可包括管线）
</li>
<li>Pipeline——多段管线，计算通过多段管线或环形空间的压降或传热量。如具有多端不同直径或标高的管线</li>
</ul>
&sect; 换热器
<ul>
<li>Heater——加热器或冷却器，计算单股物流的热力学状态，如加热器、冷却器、仅涉及压力的泵、无需知道功率的阀门或压缩机
</li>
<li>HeatX——两股物流换热器，模拟两股物流的换热过程，如管壳式换热器、空冷器、板式换热器
</li>
<li>MHeatX——多股物流换热器，模拟多股物流的换热过程，如LNG（液化天然气）换热器
</li>
<li>HxFlux——传热计算，进行散热器和热源之间的对流传热计算，如双面单层换热器</li>
</ul>
&sect; 塔设备
<ul>
<li>DSTWU——使用Winn-Underwood-Gilliland方法的多组分精馏的简捷设计模块，用于确定最小回流比、最小理论板数、实际回流比、实际理论板数等，适用于仅有一股进料和两股产品的简单精馏塔
</li>
<li>Distl——使用Edmister方法的多组分精馏简捷校核模块，用于计算产品组成，适用于仅有一股进料和两股产品的简单精馏塔
</li>
<li>RadFrac——单个塔的两相或三相严格计算模块，用于精馏塔的严格核算和设计计算，适用于普通精馏、吸收、汽提、萃取精馏、共沸精馏、三相精馏、反应精馏等
</li>
<li>Extract——液-液萃取严格计算模块，适用于萃取塔
</li>
<li>MultiFrac——严格法多塔蒸馏模块，用于对一些复杂的多塔进行严格核算和设计计算。适用于原油常减压蒸馏塔、吸收/汽提塔组合等
</li>
<li>ScFrac——简捷法多塔蒸馏模块，用于确定产品组成和浓率，估算每个塔段理论板数和热负荷等。用于原油常减压蒸馏塔
</li>
<li>PetroFrac——石油蒸馏模块，用于对石油炼制工业中的复杂塔进行严格核算和设计计算，适用于预闪蒸塔、原油常减压蒸馏塔、催化裂化主分馏塔、乙烯装置初镏塔和急冷塔组合等
</li>
<li>RateFrac——非平衡级速率模块，用于精馏塔的严格核算和设计计算，适用于蒸馏塔、吸收塔、汽提塔、共沸精馏、反应精馏等</li>
</ul>
&sect; 反应器模块
<ul>
<li>RStoic——化学计量反应器，规定反应程度和转化率的化学计量反应器模块，适用于反应动力学数据未知或不重要，但化学计量系数和反应程度已知的反应器
</li>
<li>RYield——产率反应器，规定产率的反应器模块，适用于化学计量系数和反应动力学数据未知或不重要，但产率分布已知的反应器
</li>
<li>REquil——平衡反应器，通过化学计量系数计算实现化学平衡和相平衡，适用于化学平衡和相平衡同时发生的反应器
</li>
<li>RGibbs——吉布斯反应器，通过Gibbs自由能最小实现化学平衡和相平衡，适用于化学平衡和相平衡同时发生的反应器，对固体溶液和汽-液-固系统计算相平衡
</li>
<li>RCSTR——全混釜反应器，模拟全混釜反应器，适用于带反应速率控制和平衡反应的单相、两相或三相全混釜反应器
</li>
<li>RPlug——平推流反应器，模拟平推流反应器，适用于带反应速率控制的单相、两相或三相平推流反应器
</li>
<li>RBatch——间歇式反应器，模拟间歇式或半间歇式反应器，适用于带反应速率控制的单相、两相或三相间歇和半间歇的反应器</li>
</ul>
2、分流器模块
<ul>
<li>Blocks → Specifications，Split Fraction 指本股出料与进料的摩尔流率之比；
</li>
<li>Flow 中 Key Comp NO 为预指定，在下一个 Key Components 选项卡中作为标识</li>
</ul>
3、几个注意点
<ul>
<li>饱和蒸汽，Vapor Fraction=1
</li>
<li>饱和液体，Vapor Fraction=0
</li>
<li>冷凝器，冷凝物流 汽/液比=1:1，则对应 Vapor Fraction=0.5</li>
</ul>
&sect; Heater<br />Blocks → [Heater] → Input → Speifications页面中，指定压力时，指定值>0，代表出口绝对压力；指定值<0，代表出口相对于进口的压降；<br />&sect; HeatX 模块
<ul>
<li>计算变量及使用准则
</li>
<li>[]-HT：热流体走管程
</li>
<li>[]-HS：热流体走壳程
</li>
<li>运算模式 Type：
<ul>
<li>设计 Design——简单物料平衡和能量平衡；</li>
<li>校核 Rating——根据换热要求计算换热面积；可通过膜系数法计算出K值</li>
<li>模拟 Simulation——根据实际换热面积计算两股物流的出口状态</li>
</ul></li>
</ul>
&sect; DSTWU 模块
<ul>
<li>Blocks → [DSTWU] → Input → Speifications页面中，Reflux ratio输入值>0，代表实际回流比；输入值<0，代表实际回流比与最小回流比的倍数；本模块参数中，一般选填此项；
</li>
<li>Key Component Recoveries 的含义：
</li>
<li>Light Key——轻关键组分回收率，指塔顶轻组分的量/进料中轻关键组分的量（质量或摩尔量）；
</li>
<li>Heavy Key——重关键组分回收率，指塔顶重组分的量/进料中重关键组分的量（质量或摩尔量）；
</li>
<li>模块参数设置，需在 Calculation Options 中选择 Generate table of reflux ratio vs number of theoretical stages 项，才可在结果中 Results → Reflux Ratio Profile 选项卡看到回流比随理论板数变化表。绘制曲线后，合理的理论板数应在曲线斜率绝对值较小的区域内选择（或选择上述曲线斜率变化最快/二阶导数值最大的点）。</li>
</ul>
&sect; Distl 模块
<ul>
<li>全局设定并规定报告形式，Setup → Report Options → Stream → Fraction basis 中勾选 Mole 和 Mass</li>
</ul>
&sect; RadFrac 模块
<ul>
<li>Blocks → [B1] → Setup → Streams，气相自塔底进入，可选 Above-Stage，塔板数设为 N+1
</li>
<li>Blocks → [B1] → Setup → Pressure，View 选 Top/Bottom 时，需提供塔顶冷凝器压力（全塔无压降），塔顶第一块塔板的压力（有压降），或压降（整塔压降或单板压降）</li>
</ul>
